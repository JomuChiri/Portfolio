About Me

I am the founder of Candor Labs, my personal security research and development initiative. Under Candor Labs, I am actively building and evolving multiple ongoing projects that combine security operations, detection engineering, and AI-driven automation.

Candor Labs – Web Security MVP focuses on detecting and mitigating high-impact web and AI threats, including prompt injection in AI-powered applications, SQL injection in APIs and microservices, and insecure deserialization. The project includes inline protection mechanisms, a shift-left GitHub Action scanner, and fully explainable detections with structured JSON logs, reasoning, and remediation guidance.

My second major ongoing project under Candor Labs is SOC AI Workflow — an AI-augmented SOC pipeline that integrates Elastic SIEM for detection, Python-based automation, Agentic AI for intelligent alert enrichment and triage, and Jira for streamlined case management. The aim is to significantly reduce manual workload while improving detection accuracy and response speed.

These hands-on projects allow me to deeply explore modern security challenges. I also maintain and expand a home SOC lab environment using Wazuh, Sysmon, and pfSense to support realistic attack simulation, detection testing, and broader security operations practice.
