# Digital Forensics & Malware Analysis

**Part of Candor Labs** | Ongoing Project

This section documents my hands-on work in digital forensics, focusing on memory analysis and malicious document triage within my home SOC lab environment.

## Forensics Activities

- **Memory Forensics**: Analysis of process memory dumps using Volatility to detect credential dumping, injected code, and hidden processes.
- **Malicious Document Analysis**: Static and dynamic analysis of weaponized documents (VBA macros, Excel 4.0 macros, and MSHTML exploits).
- **Tooling**: Volatility 3, Sysmon logs, ProcMon, oletools, CyberChef, and Flare-VM analysis environment.

## Available Case Studies

- [Memory Analysis Report](./memory-analysis.md) — Volatility-based memory forensics on lsass and other processes
- [Malicious Document Triage](./malicious-doc.md) — Analysis of suspicious Office documents and macro-based attacks

## Skills Demonstrated

- Memory acquisition and analysis
- Malware reverse engineering basics
- Artifact extraction from memory dumps
- Malicious document deconstruction (macros, embedded objects)
- Chain-of-custody and documentation practices

This work complements my detection engineering and incident response activities by providing deeper visibility into post-exploitation techniques.
