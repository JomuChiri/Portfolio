# SOC Lab Network Topology

**Part of Candor Labs** | Ongoing Project

A realistic, segmented home SOC lab built for safe attack simulation, detection testing, and security operations practice.

### Key Features
- pfSense firewall with strict egress rules and network segmentation
- Isolated high-risk VMs (Metasploitable & Flare-VM with no internet access)
- Static IP planning and controlled traffic flows
- Foundation for Sysmon, Wazuh, and detection rule testing
