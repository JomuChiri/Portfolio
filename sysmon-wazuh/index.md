# Sysmon + Wazuh Integration

**Part of Candor Labs** | Ongoing Project

Integrated Sysmon for rich Windows endpoint telemetry with Wazuh as the central SIEM for log collection and correlation.

### What Was Implemented
- Custom Sysmon configuration for high-value event logging
- Wazuh agent deployment on Windows endpoints
- Log forwarding and parsing from Sysmon to Wazuh Manager
- Basic correlation rules and dashboard visibility
